# Автоматизация чата под Caliban (WoT Blitz / Lesta Games)

Скрипт на Python для низкоуровневой UI-автоматизации в игре Tanks Blitz. При нажатии ЛКМ скрипт мгновенно открывает боевой чат Блица, отправляет сообщение «ВЫСТРЕЛ!» и производит огонь из орудия танков крупного калибра.

## Процесс созидания и разработки (AI-Driven Development)
* **Архитектура и логика:** Полностью спроектированы мной на основе игровых таймингов и механики чата Tanks Blitz.
* **Написание кода:** Код генерировался через ИИ под моим жестким контролем, так как я нахожусь в процессе изучения синтаксиса Python.
* **Тестирование и дебаггинг:** Первая версия кода от ИИ была сломана (отсутконфликтовали библиотеки последовательность кода). Я перенес код в VS Code, протестировал на полигоне с помощью утилит системного мониторинга, выявил ошибки в логике WinAPI и заставил ИИ пошагово исправить код до 100% рабочего состояния.

## Зависимости и требования
> **ВАЖНО:** Для работы макроса требуется библиотека `keyboard`. 
> Установка через командную строку (CMD) или терминал: `pip install keyboard`
> **Запуск:** Строго от имени Администратора (требуется Windows WinAPI для перехвата клавиш в окне DirectX игры Tanks Blitz).

## 🎛 Инструкция по настройке управления (По умолчанию)
Для корректной работы скрипта необходимо выставить следующие клавиши в настройках управления самой игры Tanks Blitz:
1. **Активация (открытие) чата** — установить на клавишу **NUM 2**.
2. **Физический выстрел орудия** — установить на клавишу **NUM 8**.
3. **Предохранитель (Запуск / Остановка макроса)** — зашит на клавишу **NUM 5** (выводит статус АКТИВЕН/ЗАБЛОКИРОВАН в консоль).

### ⚙️ Кастомизация без NUM-пада:
Если на вашей клавиатуре отсутствует NUM-раскладка (TKL / 60% клавиатуры), вы можете легко переназначить управление под себя. Для этого в файле `калибан.py` в самом начале кода изменены переменные виртуальных кодов клавиш (`VK_NUMPAD2`, `VK_NUMPAD5`, `VK_NUMPAD8`) на любые удобные вам кнопки. В коде для этого оставлены соответствующие текстовые предупреждения и комментарии.
